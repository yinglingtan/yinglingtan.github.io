Xfers Mobile App Conceptualization

Xfers is a payment gateway that makes it easy for anyone with an online bank account to collect payments and make payouts automatically.

After being an addicted user of Venmo, I'm stoked that Xfers is here in Singapore to make our lives easier. Since Xfers currently runs on web, this gave me an idea to conceptualize a mobile app.

Disclaimer: I don't have a working relationship with Xfers, this is a self-initiated project.

Looking at the Web App

For this project, I chose to focus on the dashboard and send money function.

Wireframing for Mobile

Picking out the essential functions, I designed the app by giving it a simple and clean layout with reference to Google Material Design guidelines.

Designing User Interface

The colorful scheme of the user interface is influenced by the brand color. As this is a payments app, it is important for the interface to portray a serious and secure tone. However, if Xfers develops more social elements, the interface could be more fun and lively.
